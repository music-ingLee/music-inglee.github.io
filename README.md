# music-ingLee.github.io

Personal page built for ComP Lab 12 (Vibe Coding with GitHub Pages) / HW6.

**Live:** https://music-inglee.github.io

## Structure
```
├── index.html      # entry page (all sections + nav)
├── styles.css      # dark modern theme
├── DESIGN.md       # design notes
└── assets/
    ├── avatar-placeholder.svg   # shown until you add profile.jpg
    └── (add) profile.jpg, cv.pdf
```

## To make it yours
Search `index.html` for `[ ]` and `TODO:` markers and replace with your real
name, intro, projects, courses, and contact links. Then drop `profile.jpg`
and `cv.pdf` into `assets/`.
